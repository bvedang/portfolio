## What it does

Input: GitHub repo. Output: predicted stars, forks, issues.

## Architecture

Three microservices, each a different forecasting model:

- **LSTM** — neural network, best for short-term
- **Prophet** — handles seasonality well
- **Statsmodels** — traditional stats baseline

Each runs in Docker. React frontend compares predictions side by side.

## Results

No single model wins every case. LSTM: short-term. Prophet: seasonal patterns. Combined: 85% accuracy.
