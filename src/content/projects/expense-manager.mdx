## What it does

Log expenses. See charts by category. Compare months.

## Features

- **Log expenses** with category, date, and notes
- **Category charts** show where money goes
- **Monthly view** to spot trends

## Stack

React + Material-UI frontend. Flask API. SQLite storage. Chart.js for visualization.
